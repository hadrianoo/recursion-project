# Recursion

JavaScript implementation of a merge sort and Fibonacci sequence algorithm with full test coverage.

## Installation

npm install

## Run tests

npm test

## Built with

JavaScript, Jest

## What I learned

- I gained a better understanding of the debugger and how it works. I used it extensively to tackle this problem
- I struggled with recursion at first, but now I can see how it simplifies code and improves readability
- How merge sort works and why it is efficient
- How to write tests before implementation (TDD approach)
